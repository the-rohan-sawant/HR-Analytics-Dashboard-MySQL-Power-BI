# HR-Analytics-Dashboard-MySQL-Power-BI

# Project Overview
This project presents an **HR Employee Distribution Analysis** using **MySQL** for data cleaning and analysis and **Power BI** for visualization. The objective is to analyze employee demographics, workforce distribution, turnover trends, and departmental insights to support HR decision-making through data-driven reporting.
